协方差

协方差(Covariance)描绘的是两个随机变量间的偏差程度。也描绘两个随机变量都偏离平均值时，是否朝同一个方向变化。[1]两个随机变量与各自期望的偏差的乘积的期望就是协方差。

Covariance，Co-与variance组成，共同的变化的性质，协变性。

假如存在两个随机变量X，Y，它们间的协方差称作 ：

协方差阵，用 表示，其中 表示随机向量中 与 间的协方差。
协方差的值可正可负，也可为0。

协方差的性质，在这以表格形式给出：

性质 说明
cov(X,Y) = cov(Y,X) 随机变量X与随机变量Y的协方差等同于随机变量Y与随机变量X的协方差，只是换了一个位置而已，对象没有变。
cov(X,a) = cov(b,X) = 0 随机变量X与常数a的协方差等于0，因为常数的期望值等于0，最终计算结果也为0。
cov(aX,bY) = abcov(X,Y) 两个随机变量均乘以某个常数，那么它们的协方差也要乘上这些常数。这个性质标明了协方差的大小会受到量纲的影响。
cov(X + Y, Z) = cov(X,Z) + cov(Y,Z) 协方差也有分配律 :)
相互独立的两个随机变量的协方差值为0 相互独立的两个随机变量协方差值为0；但是协方差值为0的两个随机变量不能证明这两个随机变量相互独立。[2]
D(X + Y) = D(X) + D(Y) + 2cov(X,Y);

D(X-Y) = D(X) + D(Y) - 2cov(X,Y)		 方差与协方差之间的关系。

协方差阵的性质：
协方差阵是一个对称半正定阵。[3]

协方差中的性质里曾提到协方差的大小会受到量纲的影响。接下来的问题是如何消去量纲的影响。[4]
消去量纲的影响的操作也被称作协方差标准化。
对标准化的方法就是除以随机变量X和Y的标准差，这样就能够标准化协方差，消去量纲的影响，就能正确地比较使用不同单位计算的协方差的值。

探讨一个问题——两个相互独立的随机变量，它们的协方差为0；如果它们的协方差为0，不能说明它们相互独立，但能说明什么呢？
协方差为0，说明没有线性关系，但是可能有其他依赖关系；
协方差大于0，说明两个随机变量偏离平均值时，是同向偏离的；
协方差小于0，说明两个随机变量偏离平均值时，是异向偏离的。

协方差擅长衡量线性关系的两个随机变量，而对于非线性关系的两个随机变量较为困难。

  1. 对于后者关于协方差的解释，可以举一个例子帮助理解：
    假设有两个随机变量，随机变量X表示学习时间，随机变量Y表示学习成绩，
    如果事实遵循 “学习时间越长，学习成绩越高” 的话，那么X-E(X)表示学习时间长于平均学习时间，Y-E(Y)表示学习成绩高于平均学习成绩。
    此时的X-E(X)与Y-E(Y)的乘积的协方差大于0，这个结果印证了事实 “学习时间越长，学习成绩越高” 。
    反之，如果事实遵循 ”学习时间越短，学习成绩越高“ 的话，那么此时的协方差就小于0，也同样印证了事实。 ↩︎
